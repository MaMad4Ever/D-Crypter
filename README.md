## Getting Started

To get a local copy up and running follow these simple steps.  

### Installation
1. Clone the repo
   ```bash
   $ git clone https://github.com/MaMad4Ever/D-Crypter.git
   ```
2. Install requirement packages.
   ```bash
   $ pip install -r ./D-Crypter/requirements.txt
   ```

### Usage
* Run D-Crypter.
   ```bash
   $ python3 Main.py
   ```
